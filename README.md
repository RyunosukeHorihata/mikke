Overview
mikkeはあなたのお気に入りのTwitterアカウントが見つかるサービスです。

## Description
検索フォームでキーワード入力及び検索すると、キーワードに関連するTwitterアカウントがフォロワー数順で表示されます。
検索結果一覧より、気になるTwitterアカウントがあればクリックしてください。クリックすると公式Twitterページへと遷移することができます。
